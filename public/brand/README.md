# Brand Assets — Required Replacements

This directory will hold approved brand files. Until supplied, the site uses accessible text-based fallbacks.

## Required assets

| File | Description | Format | Status |
|------|-------------|--------|--------|
| `sutur-wordmark.svg` | Full Sutur wordmark / logo | SVG (preferred) or PNG | NOT SUPPLIED |
| `sutur-icon.svg` | Sutur icon / mark only | SVG (preferred) or PNG | NOT SUPPLIED |
| `odoo-partner-badge.svg` | Official Odoo Partner badge/logo | SVG (preferred) or PNG | NOT SUPPLIED |

## Optional assets

| File | Description |
|------|-------------|
| `gotham-*.woff2` | Licensed Gotham web fonts (if procured) |

## Usage

Place approved files in this directory. The application references them via `/brand/<filename>`.

Do not:
- Redraw the official Sutur or Odoo marks as imitations
- Use placeholder images that look like real assets
- Claim Odoo Partner status without the approved badge
